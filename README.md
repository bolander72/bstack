# bstack

OpenClaw skills by [bolander72](https://github.com/bolander72).

## Skills

| Skill | Description |
|-------|-------------|
| [isync](skills/isync/) | Manage email via mbsync/isync — sync IMAP mailboxes to local Maildir |
| [caldir](skills/caldir/) | Manage calendars via caldir — view, create, sync events |
| [kokoro-tts](skills/kokoro-tts/) | Local text-to-speech via Kokoro-ONNX or mlx-audio — zero cost, no API keys |
| [maton-qbo](skills/maton-qbo/) | QuickBooks Online bookkeeping via Maton API gateway |
| [vault-sync](skills/vault-sync/) | Git-based workspace backup with smart commit messages |
| [ecobee](skills/ecobee/) | Ecobee thermostat control via Home Assistant API |
| [ratgdo32-disco](skills/ratgdo32-disco/) | Garage door control via ratgdo32 disco local web API |

## Install

```bash
clawhub install bolander72/bstack/skills/<skill-name>
```
