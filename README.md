# bstack

OpenClaw skills by [bolander72](https://boland.co).

## Skills

| Skill | Description |
|-------|-------------|
| [isync](skills/isync/) | Manage email via mbsync/isync — sync IMAP mailboxes to local Maildir, read/search messages offline |
| [caldir](skills/caldir/) | Manage calendars via caldir — view, create, sync events across iCloud and Google Calendar |

## Install

```bash
# Install a single skill
clawhub install bolander72/bstack/skills/isync
clawhub install bolander72/bstack/skills/caldir
```
